# ADMIN

Full access to system administration endpoints.

Endpoints:
- `GET /api/patients`
- `POST /api/patients`
- `PUT /api/patients/:id`
- `DELETE /api/patients/:id`
- `GET/POST/PUT/DELETE /api/doctors`
- `GET/POST/DELETE /api/medical-records`
- `POST/GET/PUT /api/payments/*`
- `POST/GET /api/billing/*`
- `GET/POST /api/workflows`
