MediFlow Backend API

This is a simple Express backend for the MediFlow hospital system.

Quick start:

1. cd backend
2. npm install
3. npm start

API endpoints:
- GET /api/patients
- POST /api/patients
- GET /api/appointments
- POST /api/appointments
- GET /api/medicines
- POST /api/medicines
- GET /api/doctor/patients
- GET /api/doctor/appointments
- POST /api/prescriptions
- GET /api/stats

The server persists data in `data.json`.
